# Vehicle Tracking - Driver App

A Progressive Web App (PWA) built with React and Vite for real-time vehicle tracking. This application is designed specifically for drivers to track their assigned vehicles and manage their status.

## Features

- **Real-time Tracking**: GPS location updates for active vehicles.
- **PWA Ready**: Can be installed on mobile devices for a native app experience.
- **Driver Management**: Login and manage vehicle assignments.
- **Live Maps**: View vehicle locations on an interactive map.

## Technologies Used

- **React**: Frontend framework.
- **Vite**: Build tool and development server.
- **Leaflet**: Interactive maps.
- **Tailwind CSS**: Utility-first CSS framework.
- **Vite PWA Plugin**: For PWA capabilities.

## Getting Started

### Prerequisites

- Node.js (Latest stable version)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Navigate to the project directory:
   ```bash
   cd vehicle-tracking-driver
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:

```bash
npm run dev
```

The application will be running at `http://localhost:4000`.

## Build for Production

To create a production build:

```bash
npm run build
```

The build assets will be in the `dist` folder.
